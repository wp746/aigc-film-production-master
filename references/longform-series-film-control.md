# Longform Series And Feature Control

Use this for 12/24/60-episode short-drama projects, long AIGC shorts, and 90-minute films.

## Longform Rule

Do not produce longform as a pile of scenes. Build a control system first.

## Required Bibles

- `Story Bible`: premise, theme, genre promise, audience, episode/movie engine.
- `Character Bible`: identity, desire, flaw, secret, relationships, arc state by episode/sequence.
- `World Bible`: rules, locations, social logic, visual laws, forbidden drift.
- `Asset Bible`: character/scene/prop/style codes and approved references.
- `Episode/Sequence Map`: each unit's dramatic function and cliffhanger/payoff.
- `Continuity State Log`: what changes after every episode or sequence.
- `Plant/Payoff Ledger`: setup, reminder, twist, payoff, consequence.
- `Production Priority Map`: which assets must be generated first to reduce drift.

## 60-Episode Short-Drama Structure

Use a repeatable engine:

- Episodes 1-3: premise, wound, hook, irreversible problem.
- Episodes 4-12: first revenge/romance/business/identity arc; frequent reversals.
- Episodes 13-24: larger antagonist system and relationship complications.
- Episodes 25-36: midpoint truth changes the story engine.
- Episodes 37-48: cost of winning, betrayal, moral trap, public reversal.
- Episodes 49-57: final compression, secrets collide, protagonist must change method.
- Episodes 58-60: payoff, final trap, emotional aftertaste, sequel door only if useful.

Each episode needs:

- cold open hook
- visible conflict
- reversal or reveal
- emotional beat
- cliffhanger or image echo
- production complexity rating

## 90-Minute Feature Structure

Build:

- logline
- theme
- protagonist arc
- antagonist force
- B-story
- Save-the-Cat-style 15 beats with approximate minute ranges
- 8-sequence outline
- set pieces
- final image
- AIGC production risk map

Do not start Image2/Seedance prompts until the first production block is chosen.

## Production Blocks

For long projects, only send one production block downstream at a time:

- `Pilot / EP01`
- `Sequence 01`
- `Act 1 sample`
- `Key trailer scenes`
- `Asset bible first`

This keeps cost, drift, and review manageable.
