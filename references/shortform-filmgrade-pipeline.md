# Shortform Film-Grade Pipeline

This is the default execution path for the user's current work: creative shorts, short-drama episodes, promotional ads, TVC, MV, cultural-tourism videos, and contest videos.

## Goal

Produce film-grade creative quality and model-executable prompts from any input:

```text
任意创意输入
-> 创意判断
-> 故事/广告核心
-> 类型片与风格策略
-> AIGC导演可行性
-> 资产/场景/道具/文字职责
-> Image2资产板
-> Image2故事板
-> Seedance 2.0提示词
-> 后期交付提示
```

## Non-Skippable Stages

Every full workflow must pass these stages. Keep them compact, but do not skip them:

1. `INTAKE_LOCK`: format, duration, platform, aspect ratio, style, assets, final deliverable.
2. `CREATIVE_LOCK`: premise, audience promise, protagonist/product/place engine, surprise or persuasion mechanism.
3. `GENRE_STYLE_LOCK`: primary genre/style, visual texture, sound texture, forbidden generic look.
4. `AIGC_FEASIBILITY_LOCK`: what is direct, asset-first, board-first, post-first, split, or avoid.
5. `PRODUCTION_HANDOFF_LOCK`: asset codes, scene codes, prop codes, UI/text/post duties, segment plan.
6. `PROMPT_FACTORY_LOCK`: Image2 asset prompts, Image2 storyboard prompts, Seedance prompts.
7. `POST_LOCK`: subtitles, exact text, UI overlays, sound, music, color, export specs.
8. `QA_LOCK`: story, feasibility, handoff, prompt, compliance, genre, final file checks.

## Film-Grade Shortform Standard

Do not use "cinematic" as a vague adjective. Film-grade means:

- camera movement has story motivation
- light has source and emotional logic
- production design carries character/product/place meaning
- sound is part of rhythm, not an afterthought
- every cut has a reason
- the final image or brand memory changes how the viewer remembers the opening
- AI model limits are designed around before generation

## Stage Output Compression

For speed, stages may be compact, but the logic must remain visible in the final working document:

- 1-3 sentence creative lock
- one genre/style paragraph
- one feasibility table
- one handoff matrix
- segment table
- final prompt package
