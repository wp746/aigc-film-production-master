# Black Humor Playbook

Use this for黑色幽默、荒诞现实、讽刺短片、职场/社会寓言、冷幽默广告、反套路脑洞短片。

## Audience Promise

The viewer laughs, then feels the wound. Black humor should reveal a cruel truth through ordinary behavior pushed one step too far.

## Story Moves

| Move | Function |
|---|---|
| Normal social rule | Establish ordinary reality |
| Absurd extension | Treat an insane premise as normal |
| Deadpan behavior | Characters do not overreact |
| Moral discomfort | The joke exposes a real system or desire |
| Escalation by logic | Each beat follows the absurd rule more seriously |
| Final sting | Last image makes the laugh curdle |

## Performance

- Underplay. The more absurd the world, the more serious the characters should behave.
- Avoid broad comedy faces unless the project is farce.
- Use micro-pauses, polite language, awkward silence, bureaucratic calm.
- Let objects carry the joke: stamp, queue number, ID photo, contract, shopping cart, office badge.

## Camera And Blocking

- Plain realism is often funnier than stylized comedy.
- Locked-off wide shot can make absurd behavior feel institutional.
- Close-up only when the character suppresses reaction.
- Keep composition clean and ordinary; absurdity comes from action/context.

## Rhythm

- Set the absurd rule quickly.
- Repeat it twice with escalation.
- Third repetition breaks moral comfort.
- Final image should be simple, cruel, and memorable.

## Sound

- Mundane sound is key: printer, elevator, keyboard, stamp, fluorescent hum, polite announcement.
- Music should not tell the audience "this is funny."
- Silence after a bureaucratic line can be the punchline.

## AIGC Risks

| Risk | Fix |
|---|---|
| Becomes slapstick | Underplay performance and lock realistic tone |
| Tone becomes too dark | Preserve comic surface in behavior/sound |
| Random absurd props | One rule, one repeated object |
| Text jokes fail | Put exact text in post or text plate |
| Ending feels only clever | Add human cost or moral sting |

## Downstream Prompt Strategy

- Asset boards should make ordinary world believable.
- Prop boards for repeated absurd object.
- Storyboards should mark repetition pattern and final sting image.
- Seedance prompts should specify deadpan performance, micro-pauses, and mundane sound.
- Exact bureaucratic notices, slogans, labels, or forms handled in post.

## Prompt Locks

```text
黑色幽默质感：现实空间保持普通和认真，荒诞规则被角色当作正常制度执行；表演克制、礼貌、死板；笑点来自重复逻辑和最后的道德刺痛，不使用夸张喜剧表情或随机怪诞道具。
```

```text
Black humor look: ordinary realistic space, absurd rule treated as normal procedure, restrained polite deadpan performance, humor from repeated logic and final moral sting, no exaggerated comedy faces or random weird props.
```

## Avoid

- 角色知道自己在演喜剧。
- 只荒诞不扎心。
- 结尾只反转，没有代价。
- 用字幕解释笑点。
